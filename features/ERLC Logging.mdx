> Use the ERLC Logging module to level your server up!

<Tabs>
  <Tab title="How to config ERLC Integration">
    <Steps>
      <Step title="Configure ERLC Logging">
        1. Run /config  
        2. Select the Module Menu, and navigate to ERLC Logging  
        3. Set the Logging Channels you need.  
        4. Just like that, you are done!  

        <Warning>You must have `Administrator` to use /config!</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Log Channel Info">
    - `Join Logs` → Sends ERLC Join Logs to this channel  
    - `Kill Logs` → Sends Kill Logs to the set channel  
    - `Command Logs` → Sends Command Logs to the set channel
  </Tab>
</Tabs>

<AccordionGroup>
  <Accordion title="ERLC Commands">
    - **/erlc execute** → Run an ERLC from Discord  
    - **/erlc players** → View ERLC Players  
    - **/erlc status** → View the Status of your ERLC Server
  </Accordion>
</AccordionGroup>
