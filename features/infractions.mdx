> Use the Infractions Module to manage Staff Discipline

<Tabs>
  <Tab title="How to config Infractions">
    <Steps>
      <Step title="Configure Infractions">
        1. Run /config  
        2. Select the Module Menu, and navigate to Infractions  
        3. Set the fields you need.  
        4. Just like that, you are done!  

        <Warning>You must have `Administrator` to use /config!</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Log Channel Info">
    - `Staff Role` → Role required to view all infractions of a user  
    - `Admin Role` → Role required to issue infractions  
    - `Infraction Types` → Edit the types of infractions for your server
  </Tab>
  
  <Tab title="Varibles">
  Below are all the avalible varibles to use for your embed customization!

`{infraction_id}` – The unique infraction ID
`{guild_id}` – The server ID
`{guild_name}` – The server name
`{guild_icon}` – The server icon URL
`{user_id}` – The ID of the user receiving the infraction
`{user_name}` – The username of the infracted user
`{user_mention}` – The mention of the infracted user
`{user_avatar}` – The avatar URL of the infracted user
`{issuer_name}` – The issuer's username
`{issuer_id}` – The issuer’s ID
`{issuer_mention}` – The mention of the issuer
`{issuer_avatar}` – The issuer’s avatar URL
`{type}` – The punishment type (warn, strike, suspension, etc.)
`{action}` – Always set to “Infraction”
`{reason}` – Reason for the infraction
`{notes}` – Additional notes, or “N/A”
`{timestamp}` – Discord-formatted timestamp
</Tab
</Tabs>

<AccordionGroup>
  <Accordion title="Infraction Commands">
    - **/infract** → Issue an Infraction  
    - **/infractions** → View a user’s total infractions  
    - **/infraction view** → View an Infraction based on the Case ID
  </Accordion>
</AccordionGroup>
