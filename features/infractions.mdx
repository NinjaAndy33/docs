> Use the Infractions Module to manage Staff Discipline

<Tabs>
  <Tab title="How to config Infractions">
    <Steps>
      <Step title="Configure Infractions">
        1. Run /config  
        2. Select the Module Menu, and navigate to Infractions  
        3. Set the fields you need.  
        4. Just like that, you are done!  

        <Warning>You must have `Administrator` to use /config!</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Log Channel Info">
    - `Staff Role` → Role required to view all infractions of a user  
    - `Admin Role` → Role required to issue infractions  
    - `Infraction Types` → Edit the types of infractions for your server
  </Tab>
</Tabs>

<AccordionGroup>
  <Accordion title="Infraction Commands">
    - **/infract** → Issue an Infraction  
    - **/infractions** → View a user’s total infractions  
    - **/infraction view** → View an Infraction based on the Case ID
  </Accordion>
</AccordionGroup>
