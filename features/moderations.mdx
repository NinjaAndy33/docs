> Use the Moderations module to manage in-game punishments!

<Tabs>
  <Tab title="How to config Moderations">
    <Steps>
      <Step title="Configure Moderations">
        1. Run /config  
        2. Select the Module Menu, and navigate to Moderations  
        3. Set the fields you need.  
        4. Just like that, you are done!  

        <Warning>You must have `Administrator` to use /config!</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Moderation Actions">
    - `Staff Role` → Role allowing who can use the Moderation Commands
  </Tab>
</Tabs>

<AccordionGroup>
  <Accordion title="Moderation Commands">
    - **/moderate** → Issue a Moderation to a user  
    - **/moderations** → View a user’s total Moderations
  </Accordion>
</AccordionGroup>
