> Use the Promotions Module to manage Staff Promotions

<Tabs>
  <Tab title="How to config Promotions">
    <Steps>
      <Step title="Configure Promotions">
        1. Run /config  
        2. Select the Module Menu, and navigate to Promotions  
        3. Set the fields you need.  
        4. Just like that, you are done!  

        <Warning>You must have `Administrator` to use /config!</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Log Channel Info">
    - `Destination Channel` → Channel where the Promotion embed gets sent  
    - `Staff Role` → Role that allows you to use Promotion Commands  
  </Tab>

  <Tab title="Variables">
    Below are all the available variables to use for your embed customization:

    `{promotion_id}` → View the ID of a promotion  
    `{user}` → Username of the promoted member  
    `{user_mention}` → Mention of the promoted member  
    `{user_id}` → User ID of the promoted member  
    `{user_avatar}` → Avatar URL of the promoted member  
    `{action}` → Always set to “Promoted”  
    `{notes}` → Additional notes, or “N/A”  
    `{issuer}` → Username of the issuer  
    `{issuer_mention}` → Mention of the issuer  
    `{issuer_id}` → Issuer’s user ID  
    `{issuer_avatar}` → Issuer’s avatar URL  
    `{role_name}` → Name of the role granted  
    `{role_id}` → ID of the role  
    `{role_mention}` → Mention of the role  
    `{guild_name}` → Server name  
    `{guild_id}` → Server ID  
    `{guild_icon}` → Server icon URL  
    `{timestamp}` → Discord-formatted timestamp  
  </Tab>
</Tabs>

<AccordionGroup>
  <Accordion title="Promotion Commands">
    - **/promote** → Issue a Promotion  
    - **/promotions** → View a user’s total promotions  
    - **/promotion view** → View a Promotion based on the Case ID  
  </Accordion>
</AccordionGroup>
