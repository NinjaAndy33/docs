> Use the Sessions Module to manage your ERLC Server

<Tabs>
  <Tab title="How to config Sessions">
    <Steps>
      <Step title="Configure Sessions">
        1. Run /config  
        2. Select the Module Menu, and navigate to Sessions  
        3. Set the fields you need.  
        4. Just like that, you are done!  

        <Warning>You must have `Administrator` to use /config!</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Log Channel Info">
    - `Destination Channel` → Channel where the Sessions embed gets sent  
    - `Staff Role` → Role that allows you to use Session Commands
  </Tab>
</Tabs>

<AccordionGroup>
  <Accordion title="Session Commands">
    - **/sessionboost** → Announce a player boost  
    - **/sessionend** → Announce your session has ended  
    - **/sessionfull** → Announce your session is full  
    - **/sessionstart** → Start a session  
    - **/sessionvote** → Start a session vote
  </Accordion>
</AccordionGroup>
