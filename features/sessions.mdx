> Use the Sessions Module to manage your ERLC Server

<Tabs>
  <Tab title="How to config Sessions">
    <Steps>
      <Step title="Configure Sessions">
        1. Run /config  
        2. Select the Module Menu, and navigate to Sessions  
        3. Set the fields you need.  
        4. Just like that, you are done!  

        <Warning>You must have `Administrator` to use /config!</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Log Channel Info">
    <ul>
      <li>`Destination Channel` → Channel where the Sessions embed gets sent</li>
      <li>`Staff Role` → Role that allows you to use Session Commands</li>
    </ul>
  </Tab>

  <Tab title="Variables">
    Below are all the available variables to use for your embed customization:

    <ul>
      <li>`{server_name}` → The name of the session server</li>
      <li>`{player_count}` → The current number of players</li>
      <li>`{max_players}` → The maximum allowed players</li>
      <li>`{duration}` → The duration of the session, or “N/A”</li>
      <li>`{join_key}` → The session join key</li>
      <li>`{owner_id}` → The session owner’s Roblox user ID</li>
      <li>`{co_owners}` → The formatted list of co-owners</li>
      <li>`{co_owners_raw}` → Raw co-owner data</li>
      <li>`{staff_role_mention}` → Mention of the assigned staff role, or “N/A”</li>
      <li>`{staff_role_id}` → The staff role ID, or “N/A”</li>
      <li>`{staff_role_name}` → The staff role name, or “N/A”</li>
      <li>`{date_timestamp}` → Full date/time stamp (Discord &lt;t:...:F&gt; format)</li>
      <li>`{relative_timestamp}` → Relative time (Discord &lt;t:...:R&gt; format)</li>
      <li>`{channel}` → Channel mention used for logging</li>
      <li>`{channel_id}` → The destination channel ID, or “N/A”</li>
      <li>`{user_name}` → The display name of the user running the session</li>
      <li>`{user_mention}` → Mention of the user</li>
      <li>`{user_id}` → The user’s Discord ID</li>
      <li>`{user_avatar}` → The URL of the user’s avatar</li>
    </ul>
  </Tab>
</Tabs>

<AccordionGroup>
  <Accordion title="Session Commands">
    <ul>
      <li>**/sessionboost** → Announce a player boost</li>
      <li>**/sessionend** → Announce your session has ended</li>
      <li>**/sessionfull** → Announce your session is full</li>
      <li>**/sessionstart** → Start a session</li>
      <li>**/sessionvote** → Start a session vote</li>
    </ul>
  </Accordion>
</AccordionGroup>
