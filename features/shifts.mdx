> Use the Shifts module to manage staff activity and duty tracking!

<Tabs>
  <Tab title="How to config Shifts">
    <Steps>
      <Step title="Configure Shifts">
        1. Run /config  
        2. Select the Module Menu, and navigate to Shift Management  
        3. Set the fields you need.  
        4. Just like that, you are done!  

        <Warning>You must have `Administrator` to use /config!</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Shift Actions">
    - `Staff Role` → Role allowing who can use the Shift Commands  
    - `On Duty Role` → Role that gets added when someone gets on duty  
    - `On Break Role` → Role that gets added when someone goes on break
  </Tab>
</Tabs>

<AccordionGroup>
  <Accordion title="Shift Commands">
    - **/shift manage** → Manage your shift  
    - **/shift leaderboard** → View all shifts of your server  
    - **/shift admin** → Edit a staff member's shift (Admin Only)  
    - **/shift reset** → Reset the shift leaderboard
  </Accordion>
</AccordionGroup>
