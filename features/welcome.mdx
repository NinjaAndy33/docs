> Use the Welcome Module to give members of your server a warm welcome!

<Tabs>
	<Tab title="How to config Welcome">
<Steps>
<Step title="Config Welcomer!"
1. Run /config
2. Select the Module Menu, and navigate to Welcome
3. Set the Welcome Message and Destination Channel.
4. Wait for a user to join your server!
<Warning>You must have `Administrator` to use /config!</Warning>
</Step>
<Steps>
</Tab>
<tab title="Varibles"
`{user}` - Mention the user who joined.
`{server}` Add the Server Name in the Message
</Tab>
